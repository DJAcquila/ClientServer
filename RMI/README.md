# RPC in ruby

## Requirements
- ruby (2.7.0p0)

## Instalation
Requires [Java](https://java.com/en/download/help/download_options.html)

Install Ruby
```sh
sudo apt-get install ruby-full
```
Install Maven
```sh
sudo apt-get install maven

```
## Running
### Server
```sh
./server
```

### Client
```sh
./client
```
